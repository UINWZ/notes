# 常用正则表达式 #
<font size="5"><b>
1. 数字: `.*[0-9]+.*`<br/>
2. 特殊字符: `.*[\p{Punct}]+.*`<br/>
3. 字母: `.*[a-zA-Z]+.*`<br/>
4. 汉字: `.*[\u4e00-\u9fa5]+.*`<br/>
5. 邮箱: `[_a-zA-Z0-9]+@[0-9a-zA-Z]+(\.[a-zA-Z]+)+`<br/>
6. 特殊符号: `((?=[\x21-\x7e]+)[^A-Za-z0-9])`<br/>
7. 日期(dddd-dd-dd): `\d{4}+[-]\d{1,2}+[-]\d{1,2}+`<br/>
</b></font>
