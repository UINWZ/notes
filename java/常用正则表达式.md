# 常用正则表达式 #

1. 数字: `.*[0-9]+.*`

2. 特殊字符: `.*[\p{Punct}]+.*`

3. 字母: `.*[a-zA-Z]+.*`

4. 汉字: `.*[\u4e00-\u9fa5]+.*`

5. 邮箱: `[_a-zA-Z0-9]+@[0-9a-zA-Z]+(\.[a-zA-Z]+)+`

6. 特殊符号: `((?=[\x21-\x7e]+)[^A-Za-z0-9])`

7. 日期(dddd-dd-dd): `\d{4}+[-]\d{1,2}+[-]\d{1,2}+`

8. 用户名: `^[a-zA-Z]\\w{5,17}$`
9. 密码: `^[a-zA-Z0-9]{6,16}$`
10. 手机号:`^((13[0-9])|(15[^4,\\D])|(18[0,4,5-9]))\\d{8}$`
11. 身份证:`(^\\d{18}$)|(^\\d{15}$)`
12. 验证URL:`http(s)?://([\\w-]+\\.)+[\\w-]+(/[\\w- ./?%&=]*)?`
13. 验证IP地址:`(25[0-5]|2[0-4]\\d|[0-1]\\d{2}|[1-9]?\\d)`
